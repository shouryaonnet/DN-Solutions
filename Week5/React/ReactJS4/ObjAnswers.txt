1. Explain the need and benefits of Component Life Cycle

The Component Life Cycle defines the stages a React component goes through from creation to removal. It helps manage data loading, updates, and cleanup efficiently.

2. Identify various Life Cycle Hook Methods

Some common lifecycle methods are:

constructor()
render()
componentDidMount()
componentDidUpdate()
componentWillUnmount()
componentDidCatch()
3. List the sequence of steps in rendering a component

The rendering sequence is:

constructor()
render()
componentDidMount()

Whenever the component updates:

render()
componentDidUpdate()

When the component is removed:

componentWillUnmount()
